# Contact Keeper (MERN STACK)
Login, register and delete, update and add new contacts.

## Installation

<ol>
<li>Create `.env` file with following data:
<ul>
<li>MONGODB_URI
<li>JWT_SECRET
</ul>
<li>Run `npm install`
<li>Run `npm run server` and it will start on your localhost:5000
</ol>
